# Study Blog Journal

This is my study blog journal website built with Next.js. I created this project to document what I am learning in web development while practicing GitHub, deployment, routing, static content updates, and project organization.

## Live Site

https://study-blog-journal.vercel.app/

## GitHub Repository

https://github.com/walcottlashay-FS/study-blog-journal.git

## About This Project

This site is set up like a simple blog. Each post is a short reflection on a web development topic or project I worked on during class. The goal is to keep the site updated with fresh content while also showing my progress as I learn how to build and deploy real projects.

## Running the Project Locally

To install the project dependencies:

```bash
npm install
```

To run the project locally:

```bash
npm run dev
```

Then open:

```text
http://localhost:3000
```
